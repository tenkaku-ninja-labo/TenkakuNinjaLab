# てんかく忍者ラボ・マニフェスト

## 合言葉
目的をひとつにする間、協力こそが推進力！

## 我らの誓い
- 我らは **透明性**を遊びに変える
- 我らは **現場の仲間**と共に汗を流す。
- 我らは **技術の民主化**を進め、誰もが触れられるソースとAPIを巻物として開く。
- 我らは **未来の技術**を先取りし、共創の場で試行錯誤を楽しむ。
- 我らは **物語を編む忍者**として、技術を文化に変える。
- 我らは **失敗を恐れず**、試みを記録し、次の仲間への道標とする。

## 我らの道
- 枕木スキャナ、レールボンドスキャナ、架電柱スキャナ──現場の目を広げる。
- エッジAIのログをGISに流し込み、台帳を自動生成する──記録を儀式に変える。
- GitHubを「巻物庫」とし、仲間が修行できる場を開く。

## 我らの未来
- 技術者と顧客、AIと人間、業者と現場──分け隔てなく混じり合う。
- 協力の力が推進力となり、文化は広がり続ける。
